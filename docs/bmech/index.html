bmech

A free, open-source Blender extension for generating mechanical parts — gears, springs, ratchets, bearings, and fasteners — parametric and ready to 3D print.

Download Latest Release View on GitHub

What's inside

Gears

External (spur, helical, herringbone), ring/annulus, planetary sets, bevel — plus PiP clearance built in.

Springs

Hairspring and serpentine spring generators.

Ratchets

External ratchet & pawl, internal freewheel ratchet.

Fasteners

Threaded fasteners.

Bearings

Ball bearings.

Install

  1. Download the latest mechanisms_core.zip from the Releases page.
  2. In Blender 5.1+, open Edit > Preferences > Get Extensions, click the dropdown in the top-right, and choose Install from Disk....
  3. Select the downloaded zip. The addon appears under Shift-A > Mechanisms in the 3D viewport.

Requirements

  • Blender 5.1 or newer
  • No external dependencies — pure Python + bmesh
